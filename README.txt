MATLAB Regression Models
This repository contains the trained regression models used in our study.
The models were developed using the Regression Learner app in MATLAB. After identifying the best-performing Support Vector Machine (SVM) model for each case, the “Generate Function” option was used to export the trained models into reusable MATLAB functions.
The repository contains eight MATLAB files. Each file corresponds to a specific combination of:
•	Gender
•	Predicted variable (Gf/Gc)
•	Item/domain level
These generated functions can be used directly for prediction on new data without retraining the models.
